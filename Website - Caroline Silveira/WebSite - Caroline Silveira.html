O que é a jQuery?

HISTÓRIA:

jQuery é uma biblioteca de funções em JavaScript, que interage com o HTML e tem como objetivo facilitar o uso de determinados recursos dessa linguagem de programação. Capaz de adicionar dinamismo e interatividade às páginas da web, ela torna mais simples a criação de animações, efeitos e manipulação de eventos.

A jQuery é uma biblioteca multi-plataforma, ou seja, compatível com qualquer navegador de internet. Seus principais objetivos são, entre outros, a redução do código JavaScript, utilização de plug-ins prontos de diversos desenvolvedores encontrados na web e resolução de alguns problemas de incompatibilidade entre browseres.

Essa biblioteca foi criada em 2006 pelo desenvolvedor americano John Resig, membro da Mozilla Corporation e graduado em Ciência da Computação pelo Rochester Institute of Technology, que tinha como objetivo elaborar códigos com foco na simplicidade e em resolver os problemas de incompatibilidade do JavaScript nos navegadores. Atualmente a jQuery, que tem como lema “Write less, do more” (escreva menos, faça mais), é a mais popular das bibliotecas JavaScript, sendo utilizada por cerca de 77% dos 10 mil sites mais visitados do mundo.

COMO UTILIZAR:

A jQuery é simplesmente um arquivo em Javascript (.js) e para utilizá-la basta anexar o arquivo no código da página em que será aplicado, conforme exemplo:

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

Este arquivo pode ser instalado no seu HD através do site oficial da biblioteca, ou ser carregado a partir de um arquivo remoto público disponibilizado pela Google, Microsoft ou jQuery.

Sintaxe

Ao interagir com o HTML, a jQuery consegue selecionar os os elementos no DOM (sigla em inglês para Document Object Model, que significa modelo de documento por objetos) que serão manipulados de forma direta. Basicamente, o DOM representa o modo como as tags em HTML, XHTML e XML serão organizadas e lidas pelo browser.

Visto isso, pode-se dizer que a sintaxe da jQuery é bem específica e prática. Ela é basicamente formada pelos elementos $.(seletor).(ação), de modo que:

$: operador jQuery.

seletor: Elemento HTML a ser manipulado.

ação: Função a ser executada pela jQuery com o elemento selecionado.

Funções gerais

Como dito anteriormente, através do comando $.(seletor).(ação) a jQuery consegue exercer diversas ações a partir do elemento selecionado. Esses comandos, também chamados de funções, basicamente representam o exato momento que uma determinada ação do usuário ocorre, sua sintaxe consiste praticamente em $(elemento).ação(function(){evento}). Veja abaixo algumas dessas funções:

Ready: é executada quando todos os elementos já estiverem sido carregados.

Click: é executada quando se aplica um clique com o botão do mouse sobre o elemento.

Mousedown e mouseup: enquanto a função mousedown é executada quando o botão do mouse está pressionado, a função mouseup é disparada quando o botão do mouse é liberado do pressionamento.

Mouseenter e mouseleave: a função mouseenter é realizada quando se posiciona o cursor do mouse sobre o elemento, já a função mouseleave é realizada quando o mouse sai de cima do elemento.

Keypress: função disparada quando pressionamos uma tecla do teclado.

Sumbit: executada quando o formulário é submetido.

Change: realizada quando um elemento de formulário tem seu valor alterado.

Variáveis

Variável é o local onde serão armazenados valores ou conteúdos que serão utlizados como parte de alguma função ou operação executada através da jQuery. A variáveis podem conter praticamente qualquer coisa, como dados complexos e funções completas, tornando possível executar os mais variados comandos.

Ao utilizarmos o JavaScript, que é uma linguagem francamente tipada e, consequentemente a jQuery, não precisamos declarar o tipo da variável que estamos contruindo, uma vez que a própria linguagem já faz essa identificação. A sintaxe de uma declaração de variável consiste em: var a = numeroX. É importante lembrar que como JavaScript é uma linguagem case-sensitive, caracteres maiúsculos e minúsculos fazem diferença na leitura do código.

VALOR DE UMA VARIÁVEL:

Alguns métodos tornam possível encontrar o valor de uma variável no documento HTML. Como por exemplo:

var X = document.getElementById(id).value: Retorna o valor da variável através do seu ID.

var X = document.getElementsByClassName(nomedaclass): Retorna o valor da variável um vetor de objetos com todos os elementos que possuem o nome da classe citada.

string.substring(): Atribui parte da string (sequência especial de caracteres) entre os índices inicial e final, ou até o final da string.

string.lenght(): Lê todo ocomprimento da string. É uma propriedade read-only (somente leitura).

str.indexOf(searchValue): procura determinado valor dentro de uma string.

parseInt e parseFloat: são utilizados quando é necessário analisar uma string e, se possível, converter e retornar um número do tipo inteiro (parseInt) ou do tipo real (parseFloat).

Efeitos

A biblioteca jQuery possui vários métodos geralmente utilizados para adicionar animações personalizadas para uma página da web. Veja a seguir alguns desses efeitos:

Hide/show: Este efeito oculta (hide) os elementos, caso estejam visíveis e mostra (show), caso estejam ocultos.

Animate: Este efeito usa as propriedades CSS para gerar as animações, gerando um vasto número de possibilidades. Sua sintaxe padrão é: $(seletor).animate({estilos CSS})

FadeIn/fadeOut: Oculta (fadeOut) ou mostra (fadeIn) os elementos gradativamente.

SlideUp e slideDown: Esse efeito mostra e oculta respectivamente os elementos em um efeito deslizante, para cima e para baixo.
